# One Piece Theory Generator AI

## Objective
A platform built with .NET 9 and OpenAI API that generates detailed One Piece theories based on user input.

## How to Run
1. Install .NET 9 SDK and necessary NuGet packages.
2. Set up OpenAI API key in `appsettings.json`.
3. Build and run the project with `dotnet build` and `dotnet run`.
4. Access the app through the provided URL.

## Dependencies
- **ASP.NET Core Web API 9**
- **ASP.NET Core MVC 9**
- **OpenAI API**
- **Layer Architecture**
- **Bootstrap 5.3.3**

## File Extensions
- **.cs**: C# code
- **.cshtml**: Razor views
- **.json**: Configuration files
